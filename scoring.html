Scoring will be done on a 1 to 5 scale, in whole numbers, in 4 "files".
  1. Security (Most Important)
  2. Innovation
  3. Speed
  4. Aesthetics

All scores within a file are averaged, then entries are sorted accordingly. Only in the case of a tie (equal averge score by all Judges) will the following file be considered.

Scores are independent, not "curved". For example, a judge could choose the following scale for "Security":

  1. Very Confident / Probably Secure
  2. Confident / Probably Secure
  3. Not Confident / Unsure
  4. Theoretical Attack(s)
  5. Practical Attack(s)

In the event of a complete tie, first, it will be checked if the tie can be broken by only including scores from Distinguished Judges. If so, then that will be the result. If not, then the Judges will be asked to re-score. If the tie cannot be broken, then both entries will pass. If the tie is for a position where a prize is awarded, multiple prizes will be awarded.

The administration will decide the number of entries to proceed to the next phase, and, after the Finalist Phase, will decide how many "Grand Prizes" and how many "High Prizes" to award. Each category will have at least 1 "Grand Prize" and at least 1 "High Prize".
