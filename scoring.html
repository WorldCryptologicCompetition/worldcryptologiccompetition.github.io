Scoring will be done on a 1-10 scale, in whole numbers, in 4 "files".

  1. 1) Security (Most Important)
  2. 2) Innovation
  3. 3) Speed
  4. 4) Aesthetics
Only in the case of a tie (equal averge score by all Judges) will the following file be considered. In the event of a full tie, first, it will be checked if the tie can be broken by only including scores from Distinguished Judges. If so, then that will be the result. If not, then the Judges will be asked to re-score. If the tie cannot be broken, then both entries will pass.
